This app was developed in the context of MAIS 202: Accelerated Introduction to Machine Learning Bootcamp, offered by McGill's AI society and serves as the final project to be presented at the McGill AI project fair. The project implements a Naive Bayes classifier for classifying financial news sentiment into three categories: negative, neutral, and positive.

Application
  1. Select an application method from the dropdown on the left: text or URL. A text field will appear if the ' text ' option is selected. If the ' URL ' option is selected, a URL field will appear, and the backend will gather the article's content directly.
  2. Click the ' analyze ' button to run the Naive Bayes classifier and output the result.
